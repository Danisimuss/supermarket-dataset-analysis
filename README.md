# supermarket-analysis
Анализ датасета и построение дашборда сети супермаркетов из Kaggle с помощью Excel (сводные таблицы, срезы). Ссылка на датасет: https://www.kaggle.com/datasets/theharsh2606/total-supermarket-sales-dataset

Сводные таблицы: выручка по категориям, тип клиента + пол, выручка по странам / городам
Визуализация: объёмная гистограмма, гистограмма с группировкой, круговая диаграмма
Интерактивность: добавлен срезы по способу оплаты, стране, полу

Категория Electronics приносит наибольшую выручку
Члены программы лояльности (Member) тратят меньше обычных покупателей
Страна Индия — самый крупный рынок по выручке
Способ оплаты Наличные приносит больше всего денег.
